# Growth Habitat — Web App UI Kit

Hi-fi recreation of the six core surfaces. Open `index.html` and use the sidebar (or `←/→` keys) to move between screens.

## Surfaces
- **Command Center** — daily home; one headline, current hypothesis, strongest arena, most energizing recent conversation, next experiment, warning signals, journey overview, quick actions.
- **01 Signal Profile** — who you are at your best.
- **02 Arena Explorer** — compare seven arenas across seven dimensions.
- **03 People & Conversations** — career intelligence map + Buffett 10% Test.
- **04 Opportunity Fit** — eight scoring dimensions, fit calculation, decision helper.
- **05 Patterns & Direction** — pattern recognition, evidence for/against, 7-day plan, exportable brief.

## Files
- `index.html` — host page with shell, sidebar, screen router.
- `Shell.jsx` — sidebar, topbar, page chrome.
- `Primitives.jsx` — Eyebrow, Card, ScoreRail, Chip, Button, etc.
- `CommandCenter.jsx`, `SignalProfile.jsx`, `ArenaExplorer.jsx`, `PeopleConversations.jsx`, `OpportunityFit.jsx`, `PatternsDirection.jsx` — one screen per file.
- `data.js` — sample data (the user's M365→AI transition context).
